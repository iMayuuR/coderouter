🚀 Quick Start

Up and Running in 3 Steps

From zero to coding with any AI model in under a minute using your favorite AI Agent.

1

📋 Copy Setup Prompt

Click the "Copy AI Agent Setup Prompt" button below to get complete installation instructions.

2

🤖 Send to AI Assistant

Paste the prompt into your AI assistant (Cursor, Claude Desktop, Windsurf, etc.). It will read your ai-agent-config.json, then automatically clone, install, configure dependencies, and test CodeRouter.

3

🚀 Launch CodeRouter

After setup completes, run .\run-claude.ps1 (Windows) or ./run-claude.sh (Unix) and start coding with 200+ AI models.

📁 AI Agent Configuration

This file tells the AI agent which model to use. You need to edit it manually with your OpenRouter API key and preferred model before giving it to your AI assistant.

  • Download this file and add your OpenRouter API key
  • Choose your preferred model from the supported list
  • Then give this edited file to your AI assistant along with the copy prompt
  • The AI uses this config to know what to install and launch
ai-agent-config.json 
{
  "user_preferences": {
    "api_key": "sk-or-v1-...",
    "model": "google/gemini-2.5-pro",
    "is_default": true
  },
  "agent_actions": {
    "install": "npm install",
    "launch": ".\\run-claude.ps1"
  }
}
