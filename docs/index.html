🚀 Quick Start

Up and Running in 3 Steps

From zero to coding with any AI model in under a minute using your favorite AI Agent.

1

Configure the Agent

Download the ai-agent-config.json file and add your OpenRouter API key and favorite model to it.

2

Hand it over

Give the ai-agent-config.json file to Cursor, Claude Desktop, or your IDE's AI Agent and say: "Set this up."

3

Run & Code!

The agent will run the setup. Once done, execute .\run-claude.ps1 (Windows) and start coding.

AI-Agent Ready Config

Just hand the ai-agent-config.json file to your favourite AI coding assistant — it contains all the instructions needed to auto-install everything.

  • Set your OpenRouter API key inside the file
  • Choose your favorite model from the supported list
  • The agent runs setup-agent.ps1 automatically
  • Zero manual configuration required
ai-agent-config.json 
{
  "user_preferences": {
    "api_key": "sk-or-v1-...",
    "model": "google/gemini-2.5-pro",
    "is_default": true
  },
  "agent_actions": {
    "install": "npm install",
    "launch": ".\\run-claude.ps1"
  }
}
